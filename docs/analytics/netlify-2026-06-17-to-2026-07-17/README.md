Netlify's current dashboard offered no CSV download control. The values in this
directory were transcribed from the accessible chart and table data exposed by
the dashboard. July 17 is a partial day. This residual post-migration window is
calibration evidence only; it is not representative of the blog's historical
traffic and must not be added to the static lifetime backfill.

(`/.env`, `/.git/config`, and similar paths). This is useful context when
comparing server request counts with engaged-reader analytics.
